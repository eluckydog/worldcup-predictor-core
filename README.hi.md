## ⚡ यह क्यों अलग है

ज़्यादातर फ़ुटबॉल भविष्यवाणी प्रोजेक्ट एक मॉडल से काम चला लेते हैं। यह वाला नहीं:

- **डुअल-इंजन आर्किटेक्चर** — पॉइसन और कारण मॉडल हर मैच पर प्रतिस्पर्धा करते हैं; सेलेक्टर सबसे अच्छा चुनता है
- **बाज़ार-जागरूक** — 4 स्रोतों का फ़्यूज़न (500.com, JC SP, अंतर्राष्ट्रीय, फ़ॉलबैक) स्वचालित मार्जिन हटाने और विश्वसनीयता भारांक के साथ
- **पारदर्शी अंशांकन** — विश्वास बनाम सटीकता को स्पष्ट रूप से मापते हैं। MD1: ≥0.60 थ्रेशोल्ड पर 66.7% सटीकता (9/24 मैच)
- **डिज़ाइन से पारदर्शी** — हर भविष्यवाणी में विश्वास स्कोर, इंजन चयन कारण और संभाव्यता वितरण होता है। कोई ब्लैक बॉक्स नहीं
- **दैनिक स्वचालन** — GitHub Actions दिन में 2 बार बिना मानव हस्तक्षेप के भविष्यवाणी चलाता है

**यह एक ओपन सोर्स इंजीनियरिंग प्रयोग है।** 971 ऐतिहासिक मैच, 50k मोंटे कार्लो सिमुलेशन प्रति मैच, BPD अपरिमेयता डिटेक्शन, बायेसियन विश्वास ट्रैकिंग — सब एक Python रिपॉजिटरी में। Fork करें, तोड़ें, बेहतर बनाएं।

## ✨ विशेषताएँ

- 🎯 **डुअल-इंजन सेलेक्टर** — डेटा गुणवत्ता के आधार पर हर मैच के लिए पॉइसन या कारण मॉडल गतिशील रूप से चुनता है
- 🔄 **50k MC सिमुलेशन** — गोल संभाव्यता वितरण के लिए सशर्त शाखाकरण नमूनाकरण
- 📊 **मल्टी-सोर्स ऑड्स फ़्यूज़न** — 500.com, JC SP, अंतर्राष्ट्रीय ऑड्स स्वचालित मार्जिन हटाने के साथ
- 🧠 **बायेसियन ट्रैकिंग** — टूर्नामेंट बढ़ने के साथ बीटा-द्विपद संयुग्म अपडेट
- 🔍 **अपरिमेयता डिटेक्शन** — बाज़ार विसंगतियों का पता लगाने के लिए BPD
- 📅 **ऐतिहासिक डेटा** — 1930 से सभी विश्व कप के 971 मैच
