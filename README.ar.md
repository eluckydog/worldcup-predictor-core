## ⚡ ما الذي يجعله مختلفًا

معظم مشاريع توقع كرة القدم تستخدم نموذجًا واحدًا. هذا المشروع لا يفعل ذلك:

- **هندسة محركين** — يتنافس نموذج بواسون والنموذج السببي في كل مباراة؛ ويختار المُحدد الأفضل
- **مواكب للسوق** — دمج 4 مصادر للأسعار (500.com، JC SP، الأسعار الدولية، احتياطي) مع إزالة تلقائية للهامش وترجيح حسب المصداقية
- **شفافية المعايرة** — نقيس العلاقة بين الثقة والدقة بشكل صريح. MD1: دقة 66.7% عند عتبة ثقة ≥0.60 (9/24 مباراة)
- **شفاف بالتصميم** — كل توقع يأتي مع درجة ثقة، سبب اختيار المحرك، وتوزيع احتمالي. لا صندوق أسود
- **تشغيل يومي تلقائي** — GitHub Actions ينفذ التوقعات مرتين يوميًا دون تدخل بشري

**هذه تجربة هندسية مفتوحة المصدر.** 971 مباراة تاريخية، 50 ألف محاكاة مونت كارلو لكل مباراة، كشف اللاعقلانية BPD، تتبع المعتقدات البايزي — كل ذلك في مستودع بايثون واحد. انسخه، حطمه، طوره.

## 🧠 الهيكلة

```
┌──────────────────────────────────────────────────────┐
│                   main.py (CLI)                       │
├──────────────────────────────────────────────────────┤
│                    Selector                           │
│         (Poisson ↔ Causal dual-engine gate)          │
├──────────────────────────────────────────────────────┤
│   ┌─────────────────┐   ┌─────────────────────────┐  │
│   │ Engine: Poisson  │   │   Engine: Causal        │  │
│   │ (Dixon-Coles)    │   │   (Double-ML / DAG)    │  │
│   └────────┬─────────┘   └──────────┬──────────────┘  │
│            │                        │                  │
├────────────┴────────────────────────┴──────────────────┤
│                  Monte Carlo Simulator                  │
│               (50k trials per match)                    │
├────────────────────────────────────────────────────────┤
│   Odds Fusion           │   Bayesian Update            │
│   (Multi-source odds    │   (Beta-Binomial             │

```

## ✨ الميزات

- 🎯 **مُحدد محركين** — يختار ديناميكيًا بين بواسون والسببية حسب جودة البيانات لكل مباراة
- 🔄 **50 ألف محاكاة مونت كارلو** — أخذ عينات مشروطة لتوزيعات احتمالية الأهداف
- 📊 **دمج متعدد المصادر** — أسعار 500.com، JC SP، الأسعار الدولية مع إزالة هامش تلقائية
- 🧠 **تتبع بايزي** — تحديثات مترافقة بيتا-ثنائية الحد مع تقدم البطولة
- 🔍 **كشف اللاعقلانية** — BPD لكشف الحالات الشاذة في السوق
- 📅 **بيانات تاريخية** — 971 مباراة من كل كؤوس العالم منذ 1930

## 🚀 بدء سريع

```bash
pip install -r requirements.txt

# توقع مباراة واحدة
python main.py --home "Brazil" --away "Argentina"

# مع تصحيح الأسعار
python main.py --home "Mexico" --away "South Korea" --use-odds

# التوقع اليومي الكامل
python scripts/daily_predict.py --save
```
