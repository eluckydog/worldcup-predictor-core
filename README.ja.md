# ⚽ 2026 World Cup Daily Predictor

**デュアルエンジン融合** — ポアソン回帰 + 因果推論 + FIFA準拠の予選＆トーナメントエンジン + 複数市場オッズ

**GitHub Actions で毎日自動更新 → [`predictions/`](./predictions/)**

**100% オープンソース · MIT ライセンス**

---

## ⚡ 他との違い

ほとんどのサッカー予測プロジェクトは一つのモデルで済ませますが、これは違います：

- **デュアルエンジンアーキテクチャ** — ポアソンと因果モデルが毎試合ごとに競合し、セレクターが最適な方を選択
- **市場価格統合** — 4つのオッズソース（500.com、JC SP、国際オッズ、フォールバック）を統合、自動的にジャッジを除去し信頼性で重み付け
- **キャリブレーションの透明性** — 信頼度と精度の関係を明示的に測定。MD1では信頼度≥0.60で66.7%の精度（9/24試合）
- **完全な透明性** — すべての予測に信頼度スコア、エンジン選択理由、確率分布を付与。ブラックボックスなし
- **毎日自動実行** — GitHub Actions が1日2回、人間の介入なしに予測を実行

**これはオープンソースのエンジニアリング実験です。** 971試合の歴史データ、1試合あたり5万回のモンテカルロシミュレーション、BPD非合理性検出、ベイズ信念追跡 — すべて一つのPythonリポジトリに。フォークして、壊して、改良してください。

## ✨ 機能

- 🎯 **デュアルエンジンセレクター** — 試合ごとにデータ品質に基づいてポアソンか因果モデルを動的に選択
- 🔄 **5万回MCシミュレーション** — 条件分岐サンプリングで得点確率分布を計算
- 📊 **マルチソースオッズ統合** — 500.com平均、JC SP、国際オッズを統合、自動ジャッジ除去
- 🧠 **ベイズ信念追跡** — ベータ二項共役更新でトーナメント進行とともに更新
- 🔍 **非合理性検出** — BPD（行動パターンデコーダー）で市場の異常を検出
- 📅 **歴史データ** — 1930年以降の全ワールドカップ971試合
