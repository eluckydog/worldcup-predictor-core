# 2026 世界杯预测模型 — 建模逻辑

> 最后更新：2026-06-22

---

## 一、核心哲学

**我们不预测"哪队会赢"。我们预测"概率分布"。**

从"猜胜平负"到"生成概率分布"这条线决定了所有设计：
- 每场比赛输出一组 `(H%, D%, A%)`，用 **蒙特卡洛模拟** 聚合为总预期
- 单场比赛准确率 ≠ 模型好坏。期望值聚合和校准质量才是

## 二、层次架构

```
Layer 4: 蒙特卡洛模拟 (MC aggregation)
    ↑
Layer 3: 淘汰赛对手分布 (bracket incentive)
    ↑
Layer 2: 出线形势分析 (LLM stakes)
    ↑
Layer 1: 平局校准 (draw calibration)
    ↑
Layer 0: Dixon-Coles 模型 (Poisson baseline)
```

### Layer 0 — DC 模型

**核心：** Dixon & Coles (1996) 双变量 Poisson 模型。
- 两队历史进球率 → 泊松分布 → 联合概率 grid (11×11)
- 修正因子 `ρ`：对低比分平局 (0-0, 1-1) 的额外权重
- 主场优势系数
- 基于 2002-2022 六届世界杯（288 场小组赛）历史数据训练

**局限：**
- 只能看"历史进球率"，**看不到**积分榜、出线形势、对方换人风格、天气、球场
- 平局概率 dp 通常在 20-30% 之间，很少有比赛 dp > max(H%, A%)
- 在该模型的信号强度下，单场平局近似随机（1/3 伯努利过程）

### Layer 1 — 平局校准

**问题：** DC 模型原始 dp 偏低，即使 MC 采样也低估真实平局率。

**方案 A — Uniform Draw Bonus（已验证）**
- 对每场的 Draw 概率加一个固定 Δ：`dp_cal = dp + Δ`
- H 和 A 按比例压缩：`(1-dp_cal) × H/(H+A)`，`(1-dp_cal) × A/(H+A)`
- **网格搜索 2002-2022 六届 288 场 → 全局最优 Δ = +10%**
- 回测准确率基线 62.8% → 校准后 63.9%（+1.1pp）
- 逐届最优 Δ：2002(+25%), 2006(+15%), 2010(+17%), 2014(+9%), 2018(0%), 2022(+10%)
- 加权平均 Δ = 12.7%，但为通用性保留 +10%

**方案 B — Elo 分桶校准（已放弃）**
- 按 Elo 差距分桶，每个桶独立寻优
- 结果：所有桶 Δ = 0%（简化模型 dp 值域太窄，22-24%，无法校准）
- 教训：校准必须基于完整 DC 模型输出，不能提前简化

**方案 C — 因果 Logistic 回归（实验性）**
- 以历史平局为因变量，`dp_cal`、matchday、Elo_gap 为自变量
- 最优模型 `dp + matchday + gap`，NLL=135.9
- 但 dp_cal 在 2026 数据中超出训练范围（32% vs 24%），出现 extrapolate 偏差
- 结论：谨慎用于 2026，因果模型不如 uniform Δ 可靠

**验证：**
- MC 模拟 100k 次 → 校准后总平局率中位数 31.9%（vs 观察到 32.5%）
- 校准不是神奇按钮，但方向正确

### Layer 2 — 出线形势分析（LLM 情境）

**为什么需要：** DC 模型不知道"瑞士 4 分 vs 加拿大 4 分，打平齐出线"。

**实现：**
1. 解析每日已完赛比分 → 12 组实时积分榜
2. 对每场剩余 MD3 比赛，计算四类场景：

| 场景 | dp 修正 | 条件 |
|------|---------|------|
| 双方打平齐出线 | **+7~10pp** | 打平后双方分数 ≥ 其他两队最高可能分 |
| 一方需赢一方需平 | -1pp | 再打平只够一方出线 |
| 双方必须赢 | -3pp | 打平都不够 |
| 至少一队已出局/淘汰 | -2pp | 无悬念或无心恋战 |

3. **首次产生 Draw pick**（DC + 校准 + 形势后 dp 成为最大概率）：
   - 瑞士 vs 加拿大：dp 32% → 42% → Draw ✅
   - 摩洛哥 vs 海地：dp 34% → 38% → Draw ✅
   - 比利时 vs 新西兰：dp 33% → 36% → Draw ✅
   - 法国 vs 挪威：dp 28% → 38%，但法国 60% 胜率 → H ❌

**方法论启示：**
- DC 模型 + LLM 情境 = 平局预测的完整链条
- 纯统计模型漏掉的"默契球动机"可以被 LLM 看穿
- 积分榜阅读是大模型最擅长的事，不应浪费

### Layer 3 — 淘汰赛对手分布

**思路：** 小组头名 vs 第二名在 R32 面对不同的对手 → 影响争一动机 → 影响平局意愿。

**2026 R32 对阵表（来自 `core/bracket.py`）：**

| 组 | 第一 vs | 第二 vs | 差距 |
|----|---------|---------|------|
| A | C2 | B1 | 中等 |
| B | A2 (South Korea) | **D1 (USA)** | **大** |
| C | D2 | A1 | 中等 |
| D | B2 | C1 | 中等 |
| E | F2 | F1 | 相同组 |
| F | E2 | E1 | 相同组 |
| G | J2 | H2 | 中等 |
| H | I2 | G2 | 中等 |
| **I** | **3R1 (弱)** | **H1 (西班牙)** | **⚡极大** |
| **J** | **3R2 (弱)** | **G1** | **⚡极大** |
| K | L2 | L1 | 相同组 |
| L | K2 | K1 | 相同组 |

**结论：**
- I 组和 J 组的头名动机最强（避开头名对手，吃第三名）
- B 组的头名动机中等（避开 USA）
- 其他组差异不大
- **但在 DC 模型精度下，这层修正太细** — 几乎不改变任何 Pick
- 保留为理论框架，实践中优先级低于 Layer 2

### Layer 4 — 蒙特卡洛模拟

**为什么需要 MC：** 单一"预测"不包含不确定性信息。

**实现：**
- 每场 50,000 次采样（numpy 矢量化，~0.5ms/场）
- 用 `np.searchsorted` 在 C 层从概率分布 `[H%, D%, A%]` 快速采样
- 聚合所有模拟 → 总平局数的分位数分布

**40 场回测用法：**
1. 固定已完赛的 40 场比分
2. 用校准 dc 对剩余 32 场各采样 5 万次
3. 统计 100k 次锦标赛完整路径（总平局数分布）

**典型输出：**
```
P5:  19 (26.4%)   25%: 22 (30.6%)   50%: 23 (31.9%)
75%: 25 (34.7%)   95%: 28 (38.9%)   Mean: 23.4 (32.5%)
```

## 三、平局概率 vs 平局 Pick — 关键概念

这是整个模型最容易被误解的地方：

| | 平局 Pick | 平局期望 |
|--|-----------|---------|
| 定义 | dp > max(H%, A%) | sum of all dp |
| 2026 预测 | 3/72 | 23.1/72 = 32.0% |
| 含义 | "这场比赛最可能平" | "32 场加起来约出 10 场平局" |

DC 模型设计的 reality：
- 很少有比赛 dp > 35%，而 H 或 A 通常在 35-70%
- 所以 Draw Pick 天然稀少，这是正常的
- 平局不是"预测出来的"，是"概率累积出来的"
- 类似保险：你从不预测哪栋楼会着火，但你知道 32 栋里约着 10 栋

## 四、数据源与回测

| 数据集 | 用途 | 场次 |
|--------|------|------|
| 2002-2022 历史小组赛 | DC 模型训练 + MC 回测 | 288 |
| 2026 已完赛小组赛 | 实时积分榜 + 基准平局率 | 40 |
| 2026 剩余比赛 | 预测 | 32 |

**历史平局率对比：**

| 时代 | 平局率 | 说明 |
|------|--------|------|
| 1930-2022 全部 | 24.4% | 基准 |
| 1998-2022 (32队时代) | 24.7% | 可直接比较 |
| 2014-2022 (三届) | 18.8%-20.8% | 历史低位 |
| **2026** | **32.5% (当前)** | **48队时代首秀，显著偏高** |

高平局率的原因假设：
- 12 组 48 队 → 实力更分散 → 更多实力接近的比赛
- 8 个最佳第三名晋级 → 更多"保平争出线"的动机
- 样本最早两轮，MD3 可能回落

## 五、局限性

1. **平局的不可预测性：** DC dp 在平局和非平局中分布完全相同（均值 33% vs 32%），模型没有区分能力
2. **Extrapolate 风险：** 2026 的结构（12×4）和 2022（8×4）不同，历史校准可能失效
3. **淘汰赛对手的弱信号：** 第 4 层的修正太细，在当前信号强度下听不见
4. **小组赛第三轮的回落：** MD3 通常平局率最低（出线保命动力），2026 最终数据可能回落至 28-30%

## 六、未来改进方向

1. **引入赔率数据：** 赔率市场的平局赔率包含更多信号
2. **Elo 直接校准：** 不是用简化 Elo，而是用 Elo 差距分桶 + DC dp 的双变量校准
3. **淘汰赛阶段模型：** 当前仅限小组赛
4. **实时更新：** 每轮结束后自动重训校准参数
5. **第三名出线情景库：** 预计算不同小组结果配对下的晋级分数线，辅助 MD3 出线形势判断
