# ⚽ 2026 World Cup Daily Predictor

**듀얼 엔진 퓨전** — 포아송 회귀 + 인과 추론 + 멀티 소스 시장 배당률

**GitHub Actions으로 매일 자동 업데이트 → [`predictions/`](./predictions/)**

**100% 오픈 소스 · MIT 라이선스**

---

## ⚡ 차별점

대부분의 축구 예측 프로젝트는 하나의 모델로 만족합니다. 하지만 이 프로젝트는 다릅니다:

- **듀얼 엔진 아키텍처** — 포아송과 인과 모델이 매 경기마다 경쟁하고, 선택기가 최적의 엔진을 선택합니다
- **시장 인지** — 4개 배당률 소스(500.com, JC SP, 국제 배당률, 폴백)를 마진 자동 제거 및 신뢰도 가중치로 융합
- **투명한 보정** — 신뢰도 대 정확도를 명시적으로 측정합니다. MD1: ≥0.60 임계값에서 66.7% 정확도(9/24 경기)
- **설계상 투명성** — 모든 예측에 신뢰도 점수, 엔진 선택 사유, 확률 분포를 제공합니다. 블랙박스 없음
- **매일 자동 실행** — GitHub Actions이 하루 2회 인간 개입 없이 예측을 실행합니다

**이것은 오픈 소스 엔지니어링 실험입니다.** 971개의 역사적 경기, 경기당 5만 회 몬테카를로 시뮬레이션, BPD 비합리성 탐지, 베이지안 신념 추적 — 모든 것이 하나의 Python 저장소에 있습니다. 포크해서 고장내고, 더 좋게 만드세요.

## ✨ 기능

- 🎯 **듀얼 엔진 선택기** — 데이터 품질에 따라 각 경기에 포아송 또는 인과 모델을 동적으로 선택
- 🔄 **5만 회 MC 시뮬레이션** — 골 확률 분포를 위한 조건부 분기 샘플링
- 📊 **멀티 소스 배당률 융합** — 500.com, JC SP, 국제 배당률 자동 마진 제거 통합
- 🧠 **베이지안 추적** — 토너먼트 진행에 따른 베타-이항 켤레 업데이트
- 🔍 **비합리성 탐지** — 시장 이상 징후 감지를 위한 BPD
- 📅 **역사적 데이터** — 1930년 이후 모든 월드컵 971경기

## 🚀 빠른 시작

```bash
pip install -r requirements.txt

# 단일 경기 예측
python main.py --home "Brazil" --away "Argentina"

# 배당률 보정 포함
python main.py --home "Mexico" --away "South Korea" --use-odds

# 전체 일일 예측
python scripts/daily_predict.py --save
```
