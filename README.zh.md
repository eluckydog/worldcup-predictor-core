# ⚽ 2026世界杯每日预测

**双引擎预测系统**，结合经典泊松回归与因果推断，实时预测2026美加墨世界杯。

**一条命令：DC 预测 → 平局校准 → 积分榜 → 出线形势 → MC 模拟 → 完整报告**

---

## 快速开始

```bash
pip install -r requirements.txt

# 完整预测管线（72场比赛，2.5秒）
python scripts/run.py

# 含蒙特卡洛模拟（总平局数投影）
python scripts/run.py --mc --save
```

## ⚡ 项目特点

- **双引擎架构** — 每场 Poisson 与 Causal 模型竞争，选择器自动挑选最优
- **显式校准** — 基于 2002-2022 全量 MC 回测的 +10% 平局概率优化
- **出线形势分析 (LLM)** — MD3 小组情景分析（打平齐出线/必须赢/淘汰赛对手差异）
- **蒙特卡洛模拟** — 10万次模拟总平局数投影（P50=23场/32.0%）
- **多源赔率融合** — 4 数据源（500.com、JC SP、国际、兜底），自动去除赔率水分
- **FIFA 合规** — 同分规则、第三名排名、R32 对手图

## 💡 平局预测方法论

**核心洞察：** 单场"平局推荐"很少（DC 模型 dp 20-30% 无法超过 H 或 A）。平局是累积的——模型预测的是赛事级别 ~32% 的平局率，而非单场平局选择。
