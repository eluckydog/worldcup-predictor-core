## ⚠️ 开发者声明

**没有任何AI能预测足球比赛结果。**

本项目仅为统计学建模练习，并非预测工具。机器学习模型（包括本项目）可以识别历史数据中的模式，但无法考虑以下因素：
- 球员伤病与体能
- 教练战术调整
- 裁判判罚与VAR介入
- 比赛当天天气
- 球队士气与心理因素
- 足球运动固有的随机性

首轮回测45.8%的正确率，在本届世界杯高达50%爆冷率的背景下，与随机猜测无本质区别。

**请将本项目用于分析和娱乐，切勿用于任何投注决策。**

## 🧠 系统架构

```
┌──────────────────────────────────────────────────────┐
│                   main.py (CLI 主入口)                 │
├──────────────────────────────────────────────────────┤

├──────────────────────────────────────────────────────┤
│   ┌──────────────────┐   ┌────────────────────────┐  │
│   │  泊松引擎         │   │   因果引擎              │  │
│   │  (Dixon-Coles)    │   │   (Double-ML / DAG)   │  │
│   └────────┬─────────┘   └──────────┬─────────────┘  │
│            │                        │                  │
├────────────┴────────────────────────┴──────────────────┤
│              蒙特卡洛模拟器 (每场5万次模拟)               │
├────────────────────────────────────────────────────────┤
│  赔率融合 (多源赔率)       │   贝叶斯更新                │

│  数据层: 971场历史比赛 (1930-2022) + 实时赔率            │
└────────────────────────────────────────────────────────┘
```

## ✨ 功能特点

- 🎯 **双引擎选择门** — 每场比赛根据数据质量动态选择泊松或因果模型
- 🔄 **5万次蒙特卡洛模拟** — 条件分支采样，计算进球概率分布
- 📊 **多源赔率融合** — 整合500.com平均指数、竞彩SP、国际赔率，自动去抽水
- 🧠 **贝叶斯信念更新** — Beta-Binomial共轭更新，随赛事推进持续修正
- 🔍 **非理性检测** — BPD行为模式解码器，检测市场异常信号
- 📅 **历史数据库** — 1930年以来971场世界杯比赛数据

## 🏆 模型表现

| 指标 | 数值 |
|------|------|
| 首轮已完赛 | 24 场 |
| 1X2准确率 | **45.8%** (11/24) |
| 中置信度准确率 (0.50-0.69) | **57.1%** (8/14) |
| 高置信度准确率 (≥0.70) | 25.0% (1/4) |

> 📌 **备注:** 2026世界杯爆冷率异常高（约50%的比赛为平局或冷门），远超历史统计水平。

### 置信度校准 / 各阈值准确率

**排除低置信度预测后，准确率显著提升：**

| 阈值 | 保留场次 | 准确率 | 说明 |
|------|---------|--------|------|
| 全部24场 | 24 | 45.8% | 基线 |
| ≥ 0.40 | 21 | 52.4% | 剔除3场（全部错误） |
| ≥ 0.50 | 18 | 50.0% | 剔除6场（2/6正确） |
| ≥ 0.55 | 17 | 52.9% | 剔除7场（2/7正确） |
| **≥ 0.60** | **9** | **66.7%** | **最佳阈值 — 2/3准确率** |
| ≥ 0.65 | 6 | 50.0% | 剔除18场（8/18正确） |
| ≥ 0.70 | 4 | 25.0% | 最差 — 越自信越错（爆冷） |

> **关键发现：** 置信度最高的比赛（≥0.70）反而最不准——模型在巴西vs摩洛哥、沙特vs乌拉圭、日本vs荷兰等爆冷场次中最自信。推荐设置 `--confidence 0.60` 输出，可覆盖9/24场且准确率达到66.7%。

## 🧪 选择门工作原理

选择门按 **4个维度** 评分，动态选择引擎：

| 维度 | 检查内容 |
|------|---------|
| 历史数据量 | 该对阵是否有足够历史比赛记录？ |
| DAG覆盖度 | 因果图是否覆盖相关因素？ |
| 泊松拟合优度 | 进球分布是否符合泊松分布？ |
| 结构突变 | 近期结果是否改变了攻防动态？ |

**运行模式:** `classic` (仅泊松) · `causal-only` (仅因果) · `auto` (自动推荐) · `debug` (调试)

## 📈 赔率融合流程

```
原始赔率 → 去抽水 → 归一化为隐含概率
→ 按来源可信度加权 → 融合 → 转换为λ偏差因子
→ 调整每队的泊松λ值 → 运行MC模拟
```

| 数据源 | 抽水率 | 权重 |
|--------|--------|------|
| 500.com平均指数 | ~6% | 0.94 (高) |
| 竞彩SP | ~13% | 0.89 (低) |
| 国际博彩赔率 | 不等 | 动态 |
| 模拟赔率 (保底) | 无 | 仅回退时使用 |
